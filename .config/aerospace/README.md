# AeroSpace Configuration

Tiling window manager configuration for [AeroSpace](https://github.com/nikitabobko/AeroSpace).

---

### General

- Starts at login
- Tiling layout by default with 18px gaps
- Mouse follows focused window
- 5 persistent workspaces (1–5) plus named workspaces

### Named Workspaces

| Key | Workspace | Purpose |
|-----|-----------|---------|
| `alt-b` | B | Browser |
| `alt-m` | M | Music (Spotify) |
| `alt-o` | O | Video Recording (OBS) |
| `alt-n` | N | Notes (Obsidian) |
| `alt-s` | S | Social Media Hub |
| `alt-t` | T | Terminal |

### Key Bindings

| Key | Action |
|-----|--------|
| `alt-h/j/k/l` | Focus left/down/up/right |
| `alt-shift-h/j/k/l` | Move window left/down/up/right |
| `alt-1–9` | Switch to workspace |
| `alt-shift-1–9` | Move window to workspace |
| `alt-minus/equal` | Resize smaller/larger |
| `alt-shift-f` | Toggle fullscreen |
| `alt-tab` | Previous workspace |
| `alt-shift-r` | Enter resize mode |
| `alt-shift-;` | Enter service mode |

### Auto-Assign

- Ghostty opens in workspace **T** (Terminal)
- Perplexity opens in workspace **B** (Browser)
