# Claude Code Status Line Configuration

Custom status line layout for [Claude Code](https://claude.ai/claude-code).

---

### Layout

**Line 1:** Model name, context percentage, context length, git branch, git changes

**Line 2:** Current working directory, session name, skills

### Display

- Full-width layout with compact threshold at 60 columns
- Color-coded segments: cyan (model), yellow (context %), green (context length), magenta (git branch), yellow (git changes)
