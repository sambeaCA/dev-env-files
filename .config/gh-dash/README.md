# GitHub Dashboard Configuration

Configuration for [gh-dash](https://github.com/dlvhdr/gh-dash), a TUI dashboard for GitHub PRs, issues, and notifications.

---

### Sections

- **PRs:** My Pull Requests, Needs My Review, Involved
- **Issues:** My Issues, Assigned, Involved
- **Notifications:** All, Created, Participating, Mentioned, Review Requested, Assigned, Subscribed, Team Mentioned

### Features

- Diff previews via [diffnav](https://github.com/dlvhdr/diffnav)
- Lazygit integration with `g` keybinding (opens lazygit in the repo)
- Smart filtering at launch
- Auto-refresh every 30 minutes
