# Ghostty Configuration

Configuration for [Ghostty](https://ghostty.org/), a fast GPU-accelerated terminal emulator.

---

### Setup
custom theme documentation: [Ghostty Custom theme documentation](https://ghostty.org/docs/features/theme#authoring-a-custom-theme)

- **Theme:** Dracula
- Config is split into two files:
  - `core-config` — core terminal settings (currently using defaults)
  - `theme-config` — color theme selection
